# f-string

比使用 `string.format` 方便

!!! quote
    <https://docs.python.org/zh-cn/3/reference/lexical_analysis.html#f-strings>


## 基本用法

``` py
s = 1
b = "abc"
f"s={s} b={b}"  # s=1 b=abc
```


## 字符串格式化

``` py
# 正数前面显示 `+` 号
f"{123:+}"   # +123
f"{-123:+}"  # -123

# 保留小数点后几位（四舍五入）
f"{123.456:>.2f}"  # 123.46

# 数字使用 `,` 格式化（只会对小数点前的数字格式化）
f"{123456.789876:,}"  # 123,456.789876

# 左右对齐
f"{123:5}"   # '  123' 指定长度，默认向右对齐
f"{123:>5}"  # '  123' 效果相同
f"{123:<5}"  # '123  ' 向右对齐

# 指定长度前后补0
f"{123:>05}" # '00123'
f"{123:<05}" # '12300'
```
