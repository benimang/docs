    文字效果#

    标题#

    • # 一级标题
    • ## 二级标题
    • ### 三级标题
    • #### 四级标题
    • ##### 五级标题
    • ###### 六级标题

    Warning

    在 mkdocs-material 的样式中，四级标题有加粗效果，感觉不太好，不建议使用。

    Warning

    markdown 标准定义文字底部加 === 表示一级标题，文字底部加 --- 表示二级标题，阅读性很差，不建议使用。

    • 斜体

    Warning

    markdown 标准定义中支持以下划线 _ 代替 * 实现粗体和斜体，但并不建议。
