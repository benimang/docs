    文字效果⚓︎

    标题⚓︎

    • # 一级标题
    • ## 二级标题
    • ### 三级标题
    • #### 四级标题
    • ##### 五级标题
    • ###### 六级标题

    Warning

    在 mkdocs-material 的样式中，四级标题有加粗效果，感觉不太好，不建议使用。

    Warning

    markdown 标准定义文字底部加 === 表示一级标题，文字底部加 --- 表示二级标题，阅读性很差，不建议使用。

    粗体 / 斜体⚓︎

    • 粗体
    • 斜体
    **粗体**
*斜体*

    Warning

    markdown 标准定义中支持以下划线 _ 代替 * 实现粗体和斜体，但并不建议。

    高亮 / 删除 / 下划⚓︎

    • pymdownx.mark
    • pymdownx.tilde
    • pymdownx.caret
    ==pymdownx.mark==
~~pymdownx.tilde~~
^^pymdownx.caret^^

    markdown_extensions:
  - pymdownx.mark
  - pymdownx.caret
  - pymdownx.tilde

    Quote

    https://squidfunk.github.io/mkdocs-material/reference/formatting/#highlighting-text

    Critic Markup⚓︎

    • deleted
    • added
    • deletedadded
    • highlighting
    • comments
    # 注意：右花括号前面的空白不需要
{--deleted-- }
{++added++ }
{~~deleted~>added~~ }
{==highlighting== }
{>>comments<< }

    markdown_extensions:
  - pymdownx.critic

    Tip

    这个插件在块中也可以正常展示。

    Quote

    https://squidfunk.github.io/mkdocs-material/reference/formatting/#highlighting-changes
